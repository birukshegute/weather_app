About This Weather App

This weather application allows users to search for current and future weather data for any city worldwide. It provides real-time weather information and a 3-day forecast to help you plan your days better.

This project was developed as part of the ALX Software Engineering backend specialization course, specifically for the webstack portfolio project. It was created by Biruk Shegute.

We utilize the OpenWeather API to provide accurate and up-to-date weather data, and the background image is courtesy of Pixabay.com.

You can find the source code for this project on our Github Repository.
